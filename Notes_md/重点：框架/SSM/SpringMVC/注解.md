# 常用注解

## 1、 Controller

### 1.1 配置扫描注解

```xml
<contex:component-scan base-package="controller"/>
```

### 1.2 创建Controller类

```java
@Controller
public class TestController{
}
```

## 2、 RequestMapping

### 1.1 为方法配置请求连接

```java
@Controller
public class TestController{
    @RequestMapping("/index/login")
    public String login(){
        return "login";
    }
}
```
