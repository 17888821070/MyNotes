01背包问题

例题 acwing 02.01背包问题

  • 初始方案
/*
dp[i][j];表示只考虑前i种物品，背包为j时能装的最大价值
result = max(dp[n][0~V])
dp[i][j]转移方程：
因为每个物品只有1个，只有用或不用两个状态
1.不用，就是考虑前i个物品，dp[i][j]=dp[i-1][j]
2.用，就是把第i个物品放进来，那就是要从j-i的重量的位置放进来一个i物品，dp[i][j]=dp[i-1][j-Vi]+Wi
取两种情况中最大的就是当前的最优
初始化：
f[0][0] = 0;
*/

import java.util.*;
public class Main{
    public static void main(String[] args){
        Scanner scanner = new Scanner(System.in);        
        int N = scanner.nextInt();
        int V = scanner.nextInt();                
        int[][] tar = new int[N][2];        
        for(int i = 0;i<N;i++){
            for (int j = 0;j<2;j++){
                tar[i][j] = scanner.nextInt();
            }
        }        
        System.out.println(backPack(tar,N,V));        
    }    
    public static int backPack(int[][] tar,int N,int V){        
        //如果背包容量为空或者物品没有，最高价值为0
        if(N==0||V==0) return 0; 
        
        //创建带有多一行一列的记录数组。0j表示没用物品，i0表示没有背包
        int[][] dp = new int[N+1][V+1];  
        
        for(int i = 1;i<=N;i++){            
            //第i个物品的重量和价值
            int vi = tar[i-1][0];
            int wi = tar[i-1][1];
            
            for(int j = 0 ; j<=V;j++){               
                dp[i][j] = dp[i-1][j];
                if(j>=vi){
                    //使用物品时的总价值
                    int useJ = dp[i-1][j-vi]+wi;
                    int nouseJ = dp[i-1][j];                    
                    dp[i][j] = Math.max(useJ,nouseJ);                    
                }                
            }
        }
        int res = 0;
        //遍历所有背包容量为V的组合
        for(int i = 0;i<=N;i++){
            res = Math.max(dp[i][V],res);
        }        
        return res;
    }        
}
  • 优化，i轮遍历只跟i-1轮有关，只需要一个一围数组即可记录dp
/*
dp[j] :表示只选用前i中物品，背包大小为j时的最高价值
result=dp[V];
转移方程：
1. 使用第i个物品，dp[j]=dp[j-vi]+wi
2. 不使用第i个物品，dp[j]=dp[j]
dp[j]=max(dp[j-vi]+wi,dp[j]);
注意，在遍历j时，需要从后向前遍历，以保障dp[j-vi]使用的是前一轮的数据，而不是此轮产生的数据
*/

import java.util.*;
public class Main{
    public static int backPack(int[][] tar,int N,int V){        
        //如果背包容量为空或者物品没有，最高价值为0
        if(N==0||V==0) return 0;
        
        //创建带有多一行的记录数组。[j]表示只选用前i中物品时的价值
        int[] dp = new int[V+1];       
        for(int i = 1;i<=N;i++){
            int vi = tar[i-1][0];
            int wi = tar[i-1][1];
            
            //此处需要从后向前转移，不能让dp[j-vi]先变化
            for(int j = V;j>0;j--){
                if(j>=vi){
                    dp[j]=Math.max(dp[j],dp[j-vi]+wi);
                }
            }
        }
        return dp[V];        
    }        
}

完全背包问题

例题 acwing 03完全背包问题

/*
d[i]:总体积恰好是i的情况下，最大价值是多少
result=max(f[0-V])
dp[i]状态转移：
dp[i]肯定是通过填充了一个新的物品实现填满
可能的情况是dp[i] = dp[i-vj]+wj; j属于0-N
dp[i]只需要在所有的跳转情况中选择最大的
*/

import java.util.*;
public class Main{
    public static int backPack(int[][] tar,int N,int V){        
        //如果背包容量为空或者物品没有，最高价值为0
        if(N==0||V==0) return 0;         
        int[] dp = new int[V+1];        
        //遍历求解总体积恰好是i的时候的最大价值
        for(int i = 1;i<=V;i++){                    
            int max = 0;
            for(int j = N-1; j>=0; j--){
                int vj = tar[j][0];
                int wj = tar[j][1];
                
                //在i大小的背包能装下vi的物品的情况下
                if(i>=vj){
                    max = Math.max(dp[i-vj]+wj,max); 
                }   
            }
            dp[i] = max;
        }       
        return dp[V];
    }        
}

/*

dp[j] :表示只选用前i中物品，背包大小为j时的最高价值
result=dp[V];
转移方程：
1. 使用第i个物品，dp[j]=dp[j-vi]+wi
2. 不使用第i个物品，dp[j]=dp[j]
dp[j]=max(dp[j-vi]+wi,dp[j]);
注意，在遍历j时，需要从前向后遍历,从后往前遍历能保证dp[j-vi]是未升级dp时的[j-vi]那一列最大的
*/
import java.util.*;
public class Main{
    public static int backPack(int[][] tar,int N,int V){        
        //如果背包容量为空或者物品没有，最高价值为0
        if(N==0||V==0) return 0; 
        
        int[] dp = new int[V+1];
        
        for(int i = 1;i<=N;i++){
            int vi = tar[i-1][0];
            int wi = tar[i-1][1];
            for(int j = 0;j<=V;j++){
                if(j>=vi){
                    dp[j]=Math.max(dp[j],dp[j-vi]+wi);
                }
            }
        }       
        return dp[V];
    }        
}

多重背包问题

例题 acwing 04多重背包问题

/*

dp[j] :表示只选用前i中物品，背包大小为j时的最高价值
result=dp[V];
转移方程：
1. 使用第i个物品，dp[j]=dp[j-vi]+wi
2. 不使用第i个物品，dp[j]=dp[j]
dp[j]=max(dp[j-k*vi]+k+wi) k属于1到s;
01背包的高级版，要从后向往前转移k个vi，选择中间最大的
*/
import java.util.*;
public class Main{
    
    public static void main(String[] args){
        Scanner scanner = new Scanner(System.in);        
        int N = scanner.nextInt();
        int V = scanner.nextInt();                
        int[][] tar = new int[N][3];        
        for(int i = 0;i<N;i++){
            for (int j = 0;j<3;j++){
                tar[i][j] = scanner.nextInt();
            }
        }        
        System.out.println(backPack(tar,N,V));        
    }    
    
    public static int backPack(int[][] tar,int N,int V){        
        //如果背包容量为空或者物品没有，最高价值为0
        if(N==0||V==0) return 0;         
        int[] dp = new int[V+1];        
        for(int i = 1;i<=N;i++){
            int vi = tar[i-1][0];
            int wi = tar[i-1][1];
            int si = tar[i-1][2];
            for(int j = V;j>0;j--){
                for(int k=0;k*vi<=j&&k<=si;k++){
                    dp[j]=Math.max(dp[j],dp[j-k*vi]+k*wi);
                }
            }
        }       
        return dp[V];
    }        
}
