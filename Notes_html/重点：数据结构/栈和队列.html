栈和队列

栈

后入先出
栈顶：允许插入和删除

基本操作

初始化、入栈、出栈、获取、判空、求长、遍历、销毁

顺序栈

数组实现，0表示栈底、size表示栈顶

public class Stack<T>{
    T[] elements ;
    int size = 0;
    public Stack(){
        elements = (T[]) new Object[];
    }
}

链栈

链表实现，head节点表示栈顶

public class Stack<T>{
    class Node{
        T value;
        Node next;
        Public Node(T value,Node next){
            this.value = value;
            this.next=next;
        }
    }
    
    Node<T> head;
    int size = 0;
}

队列

先入先出
队头队尾

基本操作

初始化、入队、出队、获取队头、求长、判空、遍历、销毁

顺序队列

数组 front~rear存数据,队列长度为rear-front

public class Queue<T>{
    T[] elements;
    int front;
    int rear;
    public Queue(){
        elements = (T[]) new Object[];
    }
}

循环队列

数组 front~rear之间或者front~数组最后一个+数组第一个~rear
front = (front+1)%length
rear = (rear+1)%length
front指向的位置不存数据，从下一个位置开始存数据
当front=rear时表示空队，(rear+1)/length=front时表示满队
